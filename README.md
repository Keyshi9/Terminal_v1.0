# Dev Terminal System

A professional, web-based terminal emulator built with Vanilla JavaScript. This project simulates a fully functional command-line interface directly in the browser, featuring a modular architecture with distinct modes for developers, crypto enthusiasts, and network diagnostics.

## Features

- **Dev Mode**: Essential utilities including JSON formatting, UUID generation, Base64 encoding/decoding, and real-time currency conversion.
- **Crypto Mode**: Fetches live cryptocurrency prices (via Binance/CoinGecko APIs), calculates ROI, and simulates trading strategies and portfolio values.
- **Network Mode**: Includes tools for HTTP latency testing ("ping"), IP geolocation lookup, and domain WHOIS information.
- **Tools Mode**: Productivity helpers such as a secure password generator, local notes/task manager, and timers.
- **Interface**: Sleek, dark-themed UI with a retro monospace aesthetic, command history navigation, and auto-completion.

## Tech Stack

- **Core**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Storage**: LocalStorage for persisting user preferences and data.
- **APIs**: Integration with public APIs for real-time financial and network data.

## Usage

Simply open `index.html` in any modern web browser to launch the terminal. Type `help` to see the list of available commands.
